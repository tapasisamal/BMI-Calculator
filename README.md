# BMI Calculator

A simple web application to calculate Body Mass Index (BMI) based on user input of weight and height. Built using HTML, CSS, and JavaScript.

# Features
- Input weight (kg) and height (cm) to calculate BMI
- Displays BMI value and category: Underweight, Normal, Overweight
- Simple and clean UI
- Reset button to clear inputs and results

# Tech Stack
- HTML
- CSS
- JavaScript (DOM Manipulation)

# What I Learned
- Handling user input and form validation in JavaScript
- Performing calculations and dynamically updating the page
- Creating interactive UI elements
